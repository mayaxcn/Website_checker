# Website_checker
通过持续检测网站访问耗时了解网站访问质量

### 1.WebChecker参数说明
```xml
<!--数据库连接串（MYSQL 5.7）-->
<add key="mysqlConn" value="********************" />

<!--待监测网站的完整URL-->
<add key="website_URL" value="https://www.baidu.com/" />

<!--检测时间单位为（秒）-->
<add key="check_interval" value="10" />
```
